# connect-4
Connect 4 is a game where players aim to get 4 of their colored discs in a row: either vertical, horizontal, or diagonal. Players taker turns in dropping in their discs, which falls to the bottom row, and stacks on each other.


##Technologies Used
HTML

CSS

Javascript

JQuery

##Getting Started
Click on a circle to place the disc.

Players alternate turns.

Click on the 'Retry' button to reset the board.
